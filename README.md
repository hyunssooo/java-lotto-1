# 로또 🤑

###기능 목록
1. 사용자에게 금액을 입력 받아 금액 만큼의 로또를 발급한다.
    * 로또 1장의 가격은 1000원이다.
2. 로또는 6개의 각각 다른 숫자로 이루어져 있다.
    * 각 숫자는 1~45 사이의 값을 가진다. 
3. 로또의 당첨 번호와 보너스 넘버를 입력 받는다.
    * 6개 일치 -> 1등 - 2000000000원
    * 5개 일치, 보너스 번호 일치 -> 2등 - 30000000원
    * 5개 일치 -> 3등 - 1500000원 
    * 4개 일치 -> 4등 50000원
    * 3개 일치 -> 5등 5000원
4. 사용자가 구매한 로또의 통계와 수익률을 내준다.
